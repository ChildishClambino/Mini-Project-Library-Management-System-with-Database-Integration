Mini Project Library Managemt System

This is an expansion and revisit of an older code/project I had already worked on.
The big changes that have happened in this revision are that now there is database integration.
There is now a function in place that allows for user, authors, books, genres, and borrowed books to all be integrated into the database.

To begin the program have all files open and run main.py
This will allow you to access the Main Menu
You can now access Book, User, Author, and Genre Operations Menu.
Each Menu allow the abilty of adding, deleting or viewing contents of each category, with the exception of the Books Operations Menu which also allows the ability to borrow and return books.

Each category also corresponds with a table holding the values in an SQL database
